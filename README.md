# book-share
Идея сделать booksharing бумажными книгами. Люди создают профиль и создают 2 списка: список желаемого и список книг которыми они могут поделится. На карте показываются люди у которых есть книга которую вы хотите. 
Как фичу можно сделать вкладку с рекомендациями. 
Для получения книг можно использовать Google Books API
